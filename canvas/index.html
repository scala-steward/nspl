nspl canvas interaction smoke test

Drive the plot to verify each interactive surface:
  • hover — move over a point. Expect hover DataRow(...) + an unhover when the mouse leaves it.
  • click — click a point. Expect click DataRow(...).
  • shift + drag — sweep a selection rectangle. Expect select N shapes → ....
  • drag (without shift) — pans the plot inside the axis range.
  • wheel — zoom toward / away from the cursor.
  • crosshair — a dashed crosshair tracks the cursor while it hovers over either plot area. It clears when the mouse leaves the plot, and is suppressed while dragging.
The line plot below the scatter only handles drag + wheel + plot-area mousedown (plus the crosshair).
